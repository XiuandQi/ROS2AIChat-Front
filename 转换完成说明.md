## 使用方法

### 快速启动
```bash
cd vue
./start.sh
```

### 手动启动
```bash
cd vue
npm install  # 首次运行
npm run dev  # 开发模式
npm run build # 生产构建
npm run preview # 预览构建结果
```

### 项目地址
- 开发服务器: http://localhost:3000
- 预览服务器: http://localhost:4173
